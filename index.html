Previous Next

Personal project - mobile Match-3 / Roguelike game. Work in progress.

Technologies used: C#, Unity Game Engine, SQLite

Go to repository

Features:

  • Togglable game sounds (on / off)
  • Match-3 game mechanisms (board element movement, completing matches, cascade matches, various special interactions)
  • Basic perk system, to be later changed into inventory system (using SQLite) - player can pick 2 items from a draft of 5 to help them during gameplay (19 total items with various effects).
  • Combat system involving Match-3 board (player and enemy can deal damage by completing matches, applying status effects on board elements, or by directly hitting the player without moving any elements of the board.
  • Defend move (player can forgo his remaining moves in order to reduce the amount of damage taken for the next move)

Salesman-ML
Previous Next

College project - Machine-Learning powered solver of the "Traveling Salesman Problem"

Technologies: Java, JavaFX

Go to repository

Features:

  • Modifiable number of cities to travel and number of specimens per generation, from 5 to 999
  • Multithreading support (between 2 and 10 threads)
  • Progress graph, updatable in real-time
  • Random generation of path costs, and togglable symmetric path cost constraint (travel cost from A to B is equal to travel cost from B to A)
  • Results page containing winning specimens and their path.
Shape Wars
Previous Next

Thesis project - multiplayer online game with JRPG-like combat system, developed with full-stack technologies

Technologies: Java, Angular, PostgreSQL

Go to repository

Features:

  • Registration and authentication of players and administrators, using JWT authorization
  • Social platform for players - player invitation, chat, and challenging for a match
  • Combat system - players form teams of 4 characters and use their abilities (damage dealing, restoration, buffs/debuffs) to take down the enemy team.
  • AI opponent, evaluating the optimal move based on the internal scoring of board result along 5 different criteria, with limited learning capabilities
  • Level-up of characters - on completing/winning a number of matches, any given character can advance to the next level, increasing their statistics.
  • Admin panel for creating new characters and abilities
  • Automatic tooltip generation system for abilities.
